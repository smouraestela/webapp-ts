Este projeto é um laboratório completo de engenharia de qualidade, criado para estudar, explorar e integrar práticas modernas de QA em um ambiente próximo ao de produção.

* A aplicação web em TypeScript serve como base para testar automação de UI e API, performance e monitoramento. O ecossistema inclui:
* Playwright para testes E2E em TypeScript
* k6 para testes de performance e geração de métricas
* Prometheus + Grafana para observabilidade e visualização
* Docker Compose para orquestração local
* Jenkins para pipelines de CI/CD

O objetivo deste repositório é estudar arquitetura de automação, experimentação com estratégias de teste, instrumentação e integração contínua — construindo, na prática, um ambiente completo de QA moderno.
